## 설치 가이드

Ubuntu 22.04에서 OpenCV 4.11.0을 자동으로 설치하는 스크립트입니다.


### 단계별 설치
```bash
git clone https://github.com/beomsuchoi/OpenCV_Install_4.11.0.git
cd OpenCV_Install_4.11.0
chmod +x 4.11.0.sh
./4.11.0.sh
```

### 설치 확인
```bash
pkg-config --modversion opencv4
```
